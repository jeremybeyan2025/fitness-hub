# Stakd

Immersive interactive fitness website for goal-based training, preset workout plans, activity programming, a weekly planner, and a clinician-consult contact pathway.

## Included

- Premium dark Stakd landing page
- Interactive goal finder
- Preset workout library with filters
- Workout preview modals
- Weekly planner builder
- Activity plan cards
- Clinician consult contact section
- Responsive mobile layout
- GitHub Pages workflow

## Files

- `index.html`
- `styles.css`
- `app.js`
- `.github/workflows/pages.yml`
- `.nojekyll`

## Deploy

In GitHub repository settings, open Pages and select GitHub Actions as the Pages source. The included workflow deploys the static site from the main branch.
